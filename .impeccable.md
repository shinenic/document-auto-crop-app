## Design Context

### Users
Music publishers and engravers processing sheet music scans at professional volume. They work with batches of scanned pages, need precise crop control, clean B&W output, and fast multi-page PDF export with compact file sizes (CCITT G4). Speed and accuracy matter more than hand-holding.

### Brand Personality
**Fast, Efficient, Technical.** The tool should feel like a sharp instrument — responsive, no-nonsense, zero wasted space. Every pixel of UI should serve a purpose. Power users should feel at home immediately.

### Aesthetic Direction
- **Theme:** Dark mode only. Deep near-black backgrounds with subtle blue undertones for depth.
- **Accent:** Teal (#5ce0c2) — used sparingly for active states, selections, and primary actions.
- **Typography:** System fonts (-apple-system stack). No decorative fonts. Monospace for numeric values.
- **References:** Apple Preview (simplicity, native feel, image-centric), Affinity Photo (professional dark UI, dense but organized panels with clear hierarchy).
- **Anti-references:** Overly decorative or playful UIs. Heavy gradients. Rounded bubbly components. Excessive animations.

### Design Principles
1. **Content first** — The image is always the hero. UI chrome should be minimal and recede. Panels, headers, and controls exist to serve the image, not compete with it.
2. **Density without clutter** — Pack information and controls tightly (like Affinity), but maintain clear visual hierarchy through spacing, typography weight, and subtle dividers — not through color or decoration.
3. **Instant feedback** — Every interaction should feel immediate. Progressive rendering, direct canvas manipulation, and real-time previews over loading spinners wherever possible.
4. **Predictable controls** — Use standard patterns (toggles, sliders, dropdowns). No clever custom widgets that require learning. A publisher processing 50 pages shouldn't have to think about the UI.
5. **Accessible by default** — WCAG AA contrast ratios. Keyboard shortcuts for all major actions. Clear focus indicators. Functional without a mouse for power users who prefer keyboard workflows.
